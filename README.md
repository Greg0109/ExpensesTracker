# ExpensesTracker

ExpensesTracker is an online web app that allows you to keep track of your expenses and budget. Users can create accounts and different types of expenses to keep track of.

![screenshot](app/assets/images/screenshot.png)

## Built With

- Ruby v3.0.0
- Ruby on Rails v6.1.3
- Bootstrap

## Live Demo

[Heroku](https://peaceful-fortress-27562.herokuapp.com/)


## Getting Started

To get a local copy up and running follow these simple example steps.

## Prerequisites

Ruby: 3.0.0

Rails: 6.1.3

Postgres: >=13.2

yarn

### Setup

Instal gems with:

```
bundle install
```

Setup database with:

```
   rails db:create
   rails db:migrate
```


### Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
    bundle exec rspec
```
